# ||ಶಾರದಾ ಮಾ ಆರತಿ||

 ಭುವನ್
 ಸರದಾ ಮಹಿಮಾಪ್ರಂಪರ |

ಭಕ್ತೋ ಕೆ ಕಲ್ಯಾಣ್ ಕೋ
 ಧರೋ ಮಾತು ಅವತಾರ್ ||
